# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

Please report security vulnerabilities by sending an email to 
![email-address-image](https://user-images.githubusercontent.com/13570741/108488604-de9b8500-72a0-11eb-82fa-38ffa6a4e54a.gif).
We will get back to you as soon as possible.

Please note that we can only take care of general security vulnerabilities in ETF and do not take responsibility for securing public deployments.
